# Floww
Musical information library.

## Goals

General

- [x] Simple notes: time, length, velocity
- [x] Parse from midi
- [ ] Note interpolation
- [x] Some operations on flowws
- [x] Floww sheet
- [x] Serialize/Deserialize floww sheet

Streaming

- [x] Floww stream encoding
- [x] Floww stream decoding
- [x] Messages
- [x] Tracks
- [x] Simple notes
